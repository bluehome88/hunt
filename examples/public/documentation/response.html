response

Nodejs response object, extended by ExpressJS Hunt adds some locals to each response

new response()

Properties:
Name Type Description
locals.myself User | null user currently authenticated by means of http://passportjs.org
locals.development boolean set to true on development enviroment, passed to template engine
locals.staging boolean set to true on staging enviroment, passed to template engine
locals.production boolean set to true on production enviroment, passed to template engine
locals.csrf string cross site request forgery protection token for CSRF middleware
locals.css Array array of css stylesheets to export to templates
locals.js Array array of client side javascripts to export to templates
locals.flash Object flash messages used by connect-flash middleware
