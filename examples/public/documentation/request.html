Class: request

request

Nodejs request object, extended not only by ExpressJS , but also by Hunt. It have some additional objects in it

new request()

Properties:
Name Type Description
user User | null currently authenticated by means of http://passportjs.org
model model shortcut to all Hunt models
redisClient object ready to use redis client
huntEmit function use Hunt event emitter
io Object Ready to use http://socket.io object with passport.js integration
